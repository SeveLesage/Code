Définition

CSS signifie Cascading Style Sheet ou en francais feuille de style il va pouvoir se placer à trois endroit différents

  1. Style dans l'entête
  2. Style dans la balise
  3. Style dans un fichier externe

Le style dans l'entête

Pour faire du style dans l'entete on utilisera la balise style directement dans head pour faire du style on va visé un element

  1. Si element balise
  2. Si element à id
  3. Si element à class

    4. Faire une div de 200px

Le style dans la balise

Pour faire du style directement sur une balise on utilise l'attribut style

Cette façon de faire n'est pas la plus conseiller car elle vise qu'un ellement et c'est moin propre que la solution de la feuille de style externe

Le style dans un fichier externe

Cette façon de faire est la plus propre et la plus utiliser il faudras parcontre faire attention a ce que le lien soi bien faite
